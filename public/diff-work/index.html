How to Find Your Discord ID

  1. Open Discord and go to User Settings (gear icon)
  2. Navigate to Advanced in the left sidebar
  3. Enable Developer Mode
  4. Right-click on your username and select Copy ID

Keyboard Shortcuts

  • Enter - Load profile when input is focused
  • Ctrl + Enter - Load profile from anywhere
  • Alt + T - Toggle dark/light theme
  • Esc - Clear input field

No data is sent to our servers except when authenticating through Discord's official OAuth2 flow.
